A premium, high-performance travel and tour service platform built for **Manoranjan Ramjham Travel and Tour Pvt. Ltd**, based in Kathmandu, Nepal. This application provides a seamless experience for tourists to explore premium vehicle fleets, curated tour packages, and expert-led trekking adventures across the Himalayas.

## 🚀 Key Features

- **Dynamic Fleet Showcase:** Interactive catalog of 100% owned vehicles, including Toyota Fortuners, BYD Atto 3 (EVs), and luxury tourist buses.
- **Service Image Sliders:** Custom-built auto-playing image galleries for every service category.
- **Detailed Inquiry Engine:** A comprehensive 10-field inquiry form that generates professional Gmail compose URLs for direct booking.
- **Smooth Navigation:** Global hash-link scrolling logic allowing seamless transition from any page to specific service sections.
- **Responsive & Modern UI:** Built with Tailwind CSS 4 and Framer Motion for a fluid, mobile-first experience.
- **Interactive Nepal Coverage:** A visual "Our Coverage" section featuring a district-level map of Nepal as a watermark background.

## 🛠️ Tech Stack

- **Framework:** Next.js 16.2.6 (App Router)
- **Library:** React 19
- **Animations:** Framer Motion & GSAP
- **Styling:** Tailwind CSS 4 (using the latest `@tailwindcss/postcss`)
- **Icons:** Lucide React
- **Language:** TypeScript

## ⚙️ Getting Started

### Prerequisites

- Node.js 20.x or higher
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/arthurrgg8-lgtm/mr-tour-service.git
   cd mr-tour-service
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
